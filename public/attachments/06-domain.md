# domain.md

## 调度领域语义

| 业务对象 | 状态 | 表达规则 |
| --- | --- | --- |
| 任务 | 排队 | 中性状态，不提示风险 |
| 任务 | 运行中 | 正常进行中 |
| 任务 | 完成 | 成功状态 |
| 任务 | 失败 | 需要恢复动作 |
| 任务 | 超时 | 需要提示原因和后续操作 |

## 风险语义

- 高危任务映射到 `var(--warning-high)`。
- 可疑任务映射到 `var(--warning-medium)`。
- 低风险任务映射到 `var(--warning-low)` 或 `var(--info)`。

## 数据安全

- AccessKey、账号、主机 IP 默认脱敏。
- 查看明文需要显式点击。
- 明文查看行为需要记录。
